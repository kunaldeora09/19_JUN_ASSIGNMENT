• What are the new tags added in HTML5?

  1. <article> => It defines the independent or self-contained content of a webpage.
  2. <output> => It is used a container element to represent the output of a calculation or outcome of user action.
  3. <canvas> => It allows drawing graphics and animations via scripting.
  4. <svg> => It is used to draw scalable vector graphics.
  5. <audio> => It defines sound content.
  6. <embed> => It defines a container for external files/application/media.
  7. <source> => It defines multiple media resources for the media elements.
  8. <video> => It defines video content within HTML document.
